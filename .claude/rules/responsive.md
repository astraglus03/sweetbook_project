# 반응형 디자인 규칙

## 브레이크포인트 (Tailwind CSS 기준)
- `sm` (640px~): 소형 태블릿
- `md` (768px~): 태블릿
- `lg` (1024px~): 노트북/데스크톱
- `xl` (1280px~): 대형 데스크톱

## 모바일 퍼스트 원칙
- 모든 스타일은 모바일(375px)부터 작성 → `sm:`, `md:`, `lg:`로 확장
- 기본 레이아웃: 1열 세로 스택
- `md:` 이상에서 2열, `lg:` 이상에서 3열 그리드 적용

## 네비게이션
- 모바일: 하단 탭 바 (내 모임 / 둘러보기 / 알림 / 프로필) — 고정 `fixed bottom-0`
- 태블릿 이상: 상단 네비게이션 바
- 햄버거 메뉴 금지 → 하단 탭으로 주요 메뉴 항상 노출

## 레이아웃 전환 규칙

### 사이드바
- `lg:` 이상: 좌측 사이드바 표시 (사진 갤러리 필터, 프로필 메뉴)
- `md:` 이하: 사이드바 → 상단 필터 바 또는 바텀 시트

### 카드 그리드
- 모바일: 1열 풀너비 카드
- `sm:`: 2열
- `lg:`: 3열
- 사진 갤러리: 모바일 2열, `sm:` 3열, `lg:` 4~5열

### 폼 페이지 (로그인/회원가입/모임 생성)
- 모바일: 풀너비 폼, 좌측 브랜딩 패널 숨김
- `lg:`: 좌우 분할 레이아웃 (브랜딩 + 폼)

### 포토북 에디터
- 모바일: 캔버스 전체화면 + 하단 시트로 페이지 네비/사진 선택
- 태블릿: 캔버스 + 우측 패널
- 데스크톱: 3패널 (좌측 페이지 + 캔버스 + 우측 사진)

### 테이블 (주문 현황)
- 모바일: 카드형 리스트로 변환 (테이블 → 카드)
- `md:` 이상: 테이블 유지

## 터치 최적화
- 터치 타겟 최소 크기: 44x44px (모바일), 36x36px (태블릿)
- 버튼 높이: 모바일 48px, 데스크톱 40px
- 입력 필드 높이: 모바일 48px, 데스크톱 42px
- 카드 간 간격: 모바일 12px, 데스크톱 20px

## 타이포그래피 스케일
| 요소 | 모바일 | 데스크톱 |
|------|--------|----------|
| 페이지 제목 | 20px | 24px |
| 섹션 제목 | 17px | 18px |
| 본문 | 14px | 14px |
| 캡션/라벨 | 12px | 13px |
| 버튼 텍스트 | 15px | 14px |

## 이미지 최적화
- `<img>` → `loading="lazy"` 필수
- 사진 갤러리 썸네일: 모바일 200px, 데스크톱 300px
- Sharp 리사이징: 원본 외에 thumbnail(200px), medium(600px), large(1200px) 생성
- WebP 포맷 우선, fallback JPEG

## 모바일 전용 UX
- 사진 업로드: 카메라 직접 촬영 옵션 (`accept="image/*" capture="environment"`)
- 초대 참여: QR 코드 스캔 → 앱/웹 자동 연결
- 풀스크린 포토북 뷰어: 스와이프로 페이지 넘기기
- Pull-to-refresh: 사진 갤러리, 알림 목록

## Tailwind 클래스 패턴 예시
```jsx
// 카드 그리드
<div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-3 sm:gap-4 lg:gap-5">

// 사이드바
<aside className="hidden lg:block lg:w-60">
<div className="lg:hidden"> {/* 모바일 대체 UI */}

// 폼 레이아웃
<div className="flex flex-col lg:flex-row">
  <div className="hidden lg:flex lg:w-1/2"> {/* 브랜딩 */}
  <div className="w-full lg:w-1/2"> {/* 폼 */}

// 하단 탭 바 (모바일)
<nav className="fixed bottom-0 inset-x-0 bg-white border-t lg:hidden">
```
